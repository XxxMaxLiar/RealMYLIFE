
1. **`docker --version`**  
    Показывает установленную версию Docker.
2. **`docker pull <image>`**  
    Скачивает образ из Docker Hub или другого репозитория.  
3. **`docker images`**  
    Показывает список всех скачанных образов.
4. **`docker run <image>`**  
    Создает и запускает контейнер из образа.  
5. **`docker ps`**  
    Показывает список запущенных контейнеров.
6. **`docker ps -a`**  
    Показывает все контейнеры (включая остановленные).
7. **`docker stop <container_id>`**  
    Останавливает запущенный контейнер.
8. **`docker rmi <image_id>`**  
    Удаляет образ.
9. **`docker logs <container_id>`**  
    Показывает логи контейнера.
10. **`docker exec -it <container_id> bash`**  
    Позволяет зайти внутрь запущенного контейнера через терминал.
11. **`docker build -t <name> .`**  
    Создает образ из Dockerfile в текущей директории.
12. **`docker-compose up`**  
    Запускает сервисы, описанные в файле `docker-compose.yml`.
