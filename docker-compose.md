- **Уровень**: **Локальная разработка / Маленькие приложения**
    
- **Основная цель**: Docker Compose используется для управления несколькими контейнерами на одном хосте. Это удобный инструмент для локальной разработки и тестирования.
    
- **Особенности**:
    
    - Легко настроить и использовать для небольших проектов.
    - Идеален для разработки, когда нужно запустить несколько контейнеров (например, база данных, веб-приложение и кэш).
    - Не предназначен для масштабируемых продакшн-систем с высокой доступностью.

**docker-compose.yml** — это конфигурационный файл для управления несколькими контейнерами. Он описывает, как связаны несколько сервисов (контейнеров), их настройки и зависимости.
