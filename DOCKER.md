### Что такое Docker?

Docker — это инструмент для создания, доставки и запуска приложений в контейнерах. Контейнеры позволяют упаковать приложение вместе со всеми его зависимостями, чтобы оно могло работать одинаково в любой среде.

### Почему Docker полезен?

• Разработчик создает Docker Image один раз, а затем использует его для запуска приложения на любом сервере.

**Создание и использование Docker Image**
• Docker Image включает все необходимые компоненты для запуска приложения.
• Разработчик описывает инструкции для Docker, который собирает пакет.
• После установки Docker на сервер, Image запускается автоматически, без необходимости настройки.
• Docker Image обеспечивает изоляцию приложений, как в виртуальных машинах.
• Каждое приложение запускается на своей мини-операционной системе.
• **КОНТЕЙНЕР** — это запущенное приложение на основе Docker Image, изолированное от основной системы.

**Отличия контейнеров от виртуальных машин**
• Контейнеры легковесны и используют урезанную операционную систему.
• Контейнеры запускаются быстрее и используют меньше ресурсов.
• Контейнеры обеспечивают изоляцию приложений, как в виртуальных машинах, но с меньшими затратами.
